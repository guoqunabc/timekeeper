# 会议计时器 TimeKeeper

一个专业的会议演讲计时器，功能完善、界面美观，适合各种会议场景。

## ✨ 核心特性

- 🎯 **简洁界面** - 大屏幕显示，适合投屏展示
- ⏰ **精确计时** - 毫秒级精度，稳定可靠
- ⚡ **智能警告** - 最后1分钟黄色警告，超时红色强化提示
- 📝 **记录管理** - 自动保存演讲记录，支持导出CSV
- 🎨 **深色主题** - 护眼深色界面，专业会议风格
- 💾 **数据持久化** - 本地存储，刷新页面不丢失数据

## 🚀 功能亮点

### 计时功能
- 支持设置演讲者姓名
- 可自定义计时时长（分钟+秒数）
- **议程模式**：自动按顺序执行预设演讲者，无需手动切换
- 最后一分钟黄色闪烁警告
- 超时后红色动态提示，强度递增

### 记录管理
- 自动记录所有演讲时间
- 显示总时长和超时时长
- 支持删除单条记录
- 一键清空所有记录
- 导出CSV文件，便于统计分析

### 操作体验
- 键盘快捷键：空格键开始/停止，ESC键重置
- 输入验证：防止无效时间输入
- 确认对话框：防止误操作
- 响应式设计：适配不同屏幕尺寸

## 📖 使用指南

### 基本操作

#### 普通模式
1. **输入信息** - 手动输入演讲者姓名和时间
2. **开始计时** - 点击"开始"按钮或按空格键
3. **结束计时** - 点击"停止"按钮，确认结束后自动保存记录

#### 议程模式（有配置时自动启用）
1. **自动加载** - 页面加载时自动显示第一位演讲者
2. **开始计时** - 点击"开始"按钮或按空格键
3. **自动切换** - 结束计时后自动切换到下一位演讲者
4. **自动完成** - 所有演讲者完成后自动回到普通模式
5. **查看记录** - 底部显示所有演讲记录，支持删除和导出

### 预设配置
修改 `config.js` 文件来自定义预设演讲者：
```javascript
window.TIMEKEEPER_CONFIG = {
    speakers: [
        {
            name: "张三",
            minutes: 5,
            seconds: 0
        }
        // 添加更多演讲者...
    ]
};
```
- **启用议程模式**：添加演讲者到配置文件，自动按顺序执行
- **关闭议程模式**：将 `speakers: []` 设为空数组，回到普通模式
- **添加演讲者**：复制现有格式添加新对象  
- **即时生效**：修改后刷新页面即可
- **结构清晰**：配置与代码分离，便于管理

### 快捷操作
- **空格键** - 开始/停止计时
- **ESC键** - 重置计时器（仅在未开始时）

### 记录管理
- **删除记录** - 点击记录右侧的 ×  按钮
- **清空记录** - 点击"清空"按钮删除所有记录  
- **导出数据** - 点击"导出"按钮生成CSV文件

## 🛠 使用方法

### 快速开始
1. 下载项目文件
2. **直接双击 `index.html` 即可使用**（无需任何服务器或安装）

### 自定义配置
1. 用文本编辑器打开 `config.js`
2. 修改 `TIMEKEEPER_CONFIG` 对象中的演讲者列表
3. 保存文件，刷新浏览器即可生效

```

```

- ✅ **零依赖**：纯HTML/CSS/JS，无需安装任何软件
- ✅ **即开即用**：双击HTML文件直接运行
- ✅ **配置简单**：修改JS文件即可自定义
