# 会议计时器 TimeKeeper

一个专业的会议演讲计时器，功能完善、界面美观，适合各种会议场景。

## ✨ 核心特性

- 🎯 **简洁界面** - 大屏幕显示，适合投屏展示
- ⏰ **精确计时** - 毫秒级精度，稳定可靠
- ⚡ **智能警告** - 最后1分钟黄色警告，超时红色强化提示
- 📝 **记录管理** - 自动保存演讲记录，支持导出CSV
- 🎨 **深色主题** - 护眼深色界面，专业会议风格
- 💾 **数据持久化** - 本地存储，刷新页面不丢失数据

## 🚀 功能亮点

### 计时功能
- 支持设置演讲者姓名
- 可自定义计时时长（分钟+秒数）
- 最后一分钟黄色闪烁警告
- 超时后红色动态提示，强度递增

### 记录管理
- 自动记录所有演讲时间
- 显示总时长和超时时长
- 支持删除单条记录
- 一键清空所有记录
- 导出CSV文件，便于统计分析

### 操作体验
- 键盘快捷键：空格键开始/停止，ESC键重置
- 输入验证：防止无效时间输入
- 确认对话框：防止误操作
- 响应式设计：适配不同屏幕尺寸

## 📖 使用指南

### 基本操作
1. **设置时间** - 输入演讲者姓名，选择预设时间或手动输入分钟/秒数
2. **开始计时** - 点击"开始"按钮或按空格键
3. **结束计时** - 点击"停止"按钮，确认结束后自动保存记录
4. **查看记录** - 底部显示所有演讲记录，支持删除和导出

### 快捷操作
- **空格键** - 开始/停止计时
- **ESC键** - 重置计时器（仅在未开始时）

### 记录管理
- **删除记录** - 点击记录右侧的 ×  按钮
- **清空记录** - 点击"清空"按钮删除所有记录  
- **导出数据** - 点击"导出"按钮生成CSV文件
